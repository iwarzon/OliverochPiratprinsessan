# Solution Learnings — Oliver och Piratprinsessan

**Last updated:** 2026-06-24

## AI Illustration Consistency

**Problem**: AI-generated illustrations are inconsistent across generation sessions — characters change appearance between pages.

**Solution**: Create detailed character design boards (master templates) with specific visual references for every character detail. Feed these as reference material into each generation session. This grounds the AI and significantly improves cross-session consistency.

**Best practice**: Always prepare source material (character boards, story context, visual instructions) before starting illustration generation. Bind everything into a single AI project/context.

## AI Content Quality Curve

**Finding**: AI-generated content reaches ~60-70% quality quickly but pushing to 80-90%+ takes exponentially more effort. The last 30% of quality requires disproportionate iteration.

**Best practice**: Use AI for the initial draft/concept, then invest human effort in refinement. Don't expect polished output from a single generation pass.

## AI Image Refinement Technique

**Finding**: When refining AI-generated images, asking the AI to "scale up" an image first gives it a chance to create a high-fidelity clone of your reference. From there, you can address specific parts incrementally.

**Best practice**: Scale up first, then make targeted edits — don't try to fix everything in one prompt.
